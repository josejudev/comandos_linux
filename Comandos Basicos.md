# Comandos Basicos

## Remover paquetes

```
sudo apt-get --purge remove eliminar apss
sudo apt-get --remove +ppa elminar ppa
sudo apt autoremove Limpiar archivos
sudo dpkg -i package_file.deb. instalar packetes
```

## Cuando snap tienda a fallar

```
systemctl enable snapd.service
systemctl start snapd.service
```

## Archivos corruptos

```
sudo apt-get check checar dependencias de paquetes
```
